Presence & Status Flow
Online • Away • Busy • DND • Heartbeat • Auto-Away

presence:update{ status: 'online' }

PostgreSQL UPDATEusers SET status='online'
io.emit presence:changedBroadcast to all users (including self)
Every 30 Seconds
presence:heartbeatClient emits periodically
Redis REFRESHReset TTL to 5 minutes
If heartbeat stops (tab closed / crash) → Redis TTL expires → status auto-set to offline
Socket Disconnect
handleDisconnect()Server detects socket close
Redis DELRemove presence key
PostgreSQL UPDATEstatus='offline', last_seen=NOW()
Broadcastpresence:changed { status: 'offline' }
